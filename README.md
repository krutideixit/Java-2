#  Java Assignment – Vehicle Class (OOP Concepts)

##  Overview
This project implements a Vehicle Management System in Java to demonstrate core Object-Oriented Programming (OOP) concepts such as:

- Encapsulation (private members with getters/setters)
- Constructors (default & parameterized)
- Methods (start, stop, accelerate, mileage calculation)

The program creates multiple vehicle objects and compares their details and mileage.

---

## Features

- Create vehicle objects using:
  - Default constructor
  - Parameterized constructors
- Use getters and setters for private data members
- Perform vehicle operations:
  - Start
  - Drive
  - Accelerate
  - Stop
- Calculate mileage
- Display vehicle details in structured format
- Compare multiple vehicles using arrays

##  How It Works

1. Create vehicle objects using constructors
2. Set values using setters where required
3. Store vehicles in an array
4. Perform operations:
   - Start vehicle
   - Drive
   - Accelerate
   - Calculate mileage
5. Display details for each vehicle

---

##  How to Run

1. Compile the program:
   bash    javac Vehicle.java MainForVehicle.java    

2. Run the program:
   bash    java MainForVehicle
